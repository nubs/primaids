Arrays

Static utility class for working with the PHP array primitive type.

Methods

Converts the given array into a string using the provided format.

format(array $array, string $format, string $keyPlaceHolder = '{key}', string $valuePlaceHolder = '{value}') : string
static final

Example:

use Chadicus\Primaids\Arrays;
$array = ['oranges' => .69, 'bananas' => .79, 'apples' => .89];
echo Util::format($array, "Fruit: {key} only {value} per pound\n");


Output:

Fruit: oranges .69 per pound
Fruit: bananas .79 per pound
Fruit: apples .89 per pound

Arguments

$array

array

The array to be formatted.

$format

string

The format template string.

$keyPlaceHolder

string

The string the represents the place-holder within the format for the array key

                            values.

$valuePlaceHolder

string

The string the represents the place-holder within the format for the array

                            values.

Response

string

The formatted string.

Unsets and returns the value at index $key in the given array.

getAndUnset(array &$array, integer|string $key) : mixed
static final

Example: 

use Chadicus\Primaids\Arrays;
$array = ['a', 'b', 'c'];
$result = Arrays::getAndUnset($array, 1);
var_dump($result);
var_dump($array);

Output:

string(1) "b"
array(2) {
  [0] =>
  string(1) "a"
  [2] =>
  string(1) "c"
}

array

$key

integer|string

The key in $array to unset and value to be returned.

mixed

Traverses the given $array using the key path specified by $delimitedKey and returns the final value.

getNested(array $array, string $delimitedKey, string $delimiter = '.', boolean $strict = false) : mixed
static final

Example: 

use Chadicus\Primaids\Arrays;
$array = [


'db' => [
    'host' => 'localhost',
    'login' => [
        'username' => 'scott',
        'password' => 'tiger',
    ],
],


];
echo Arrays::getNested($array, 'db.login.username');

Output:

scott

throws

Thrown if $key is not an index of $array.

Arguments

$array

array

The array to traverse.

$delimitedKey

string

A string of keys to traverse into the array.

$delimiter

string

A string specifiying how the keys are delimited. The default is '.'.

$strict

boolean

Flag to throw Exception if any part of the given $delimitedKey is not found in $array.

Response

mixed

The value a the inner most key or null if a key does not exist.

Return an array containing a subset of the given $input array.

subSet(array $input, array $keys, boolean $strict = false) : array
static final

Example:

use Chadicus\Primaids\Arrays;
$full = [
    'author' => 'Gambardella, Matthew',
    'title' => 'XML Developer\'s Guide',
    'genre' => 'Computer',
    'price' => 44.95,
    'publishDate' => 970372800,
    'description' => 'An in-depth look at creating applications with XML.',
    'id' => 'bk101',
    'url' => '/books/bk101',
];
$minimal = Arrays::subSet($full, ['url', 'title', 'price']);
var_dump($minimal);


Output:

array(3) {
  'url' =>
  string(12) "/books/bk101"
  'title' =>
  string(21) "XML Developer's Guide"
  'price' =>
  double(44.95)
}
